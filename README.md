# Mobile Authentication App (firebase, expo)

This app lets users sign up and login using firebase auth

## Installation

clone the repo

```bash
  git clone https://github.com/TMohammadX/mobile-login.git
```

## Usage
```bash
  cd mobile-login
```
## Install And Run
```bash
  npm install
```
```bash
  npm start
```
